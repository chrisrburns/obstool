Welcome to Chris Burns' MWO Observing Tool

{% endblock %} {% block content %}


Here are some custom tools developed by Chris Burns, specifically to help with operating the telescopes at Mount Wilson Observatory.

  • The Interactive Nicholson List

    This tool gives both a spreadsheet-like list of currently visible objects as well as visual representation of the sky. Best viewed with a browser window at least 1400 pixels wide (or you can scale it down using your browser's controls).

  • Birthday/Anniversary Star Finder

    This is a google spreadsheet with all visible (V>12) stars within 100 ly of the Earth. It is sorted by distance (in ly), so hopefully easy to find a star that matches a specific year. It is shared read-only, so can be viewed, but if you want to filter and/or sort the data, you will need to make a copy to your own google account.

  • 60-inch Operations Manual

  • Weather Links

    • Richard Bell's Weather Station
    • The Mount Wilson Observatory Clear Sky Chart
    • National Weather Service forecast
    • HPWREN tower cameras

  • GitHub Site

    If you want to run the tool natively on your own computer, you can go here to get the code. It runs under python with only a few extra modules, so likely can run on Mac, Windows, or Linux.
